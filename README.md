# gui
嵌入式GUI框架

## 框架

```mermaid
┌─────────────────────────────────────────────────────────────────────┐
│                        应用层（Application）                          │
│                                                                     │
│  各个具体页面：主页、设置页、WiFi页 ...                                  │
└─────────────────────────────────────────────────────────────────────┘
                                  ↑ ↓
┌─────────────────────────────────────────────────────────────────────┐
│                    页面管理层（Page Manager）                          │
│                                                                      │
│  页面栈 + 生命周期管理 + 页面跳转调度                                     │
└─────────────────────────────────────────────────────────────────────┘
                                  ↑ ↓
┌─────────────────────────────────────────────────────────────────────┐
│                       GUI驱动层（GUI Driver）                         │
│                                                                      │
│  LVGL / emWin / 自研GUI库 / 裸屏驱动                                  │
└─────────────────────────────────────────────────────────────────────┘
```
